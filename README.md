Personal portfolio of **Muhammad Taha Yasin** — MERN Stack & Next.js Developer.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **GSAP** — page transition & entrance animations
- **Web3Forms** — contact form email delivery

---

## Features

- Animated loader with glitch effect
- GSAP card entrance animations on load
- Clip-path page transition when opening detail pages
- Interactive contact form (hover to reveal, sends real emails)
- Night sky animated background
- Detail pages for Projects, Skills, and About
- Fully responsive layout

---

## Getting Started

```bash
git clone https://github.com/TahaY291/portfolio.git
cd portfolio
npm install
npm run dev
```

---

## Environment Variables

Create a `.env.local` file in the root of the project:

```
NEXT_PUBLIC_WEB3FORMS_KEY=your_access_key_here
```

Get your free access key at [web3forms.com](https://web3forms.com).

> `.env.local` is gitignored and will never be pushed to GitHub.
